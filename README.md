## OverView
This project is a Java application built using a clean layered architecture (Domain/Persistence/Presentation).
It provides a command‑line interface (CLI) to manage: Products, Categories, Quantities, Inventory operations...
The goal is to demonstrate good software engineering practices, separation of concerns, and testability.
